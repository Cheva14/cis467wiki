Project Description

Background

iQuest is a mobile application that allows users with autism and other disabilities and their caregivers to directly interact with a tool called iQuest (iQuest = independence Quest). The purpose of the iQuest is to help users identify and practice independence goals they can work on at home or school.

Description of Intended Features/Backlog

User Interface:

  • Ensure the menu remains visible at all times, providing easy navigation options.
  • Consider implementing a dropdown menu with smooth animations (drawerAnimation) for a seamless user experience.
  • Be able to change the color and pictures of the user interface allowing the user to have different "themes"

Goals Management:

  • Integrate a user-friendly dropdown menu for goal categorization, allowing users to click and select various categories. (See here for list of goals)
  • Empower users to create and customize their own goals for a personalized experience.

Goal Progress Tracking:

  • Home Goal Progress feature:
    • Enable users to upload pictures to document their progress.
    • Offer visually appealing and downloadable certificates to celebrate achievements, designed with a mature yet exciting aesthetic.
    • Consider individualized certificates to make users feel special.

Goal Completion Celebrations:

  • Implement celebratory elements after goal completion.
  • Allow users to report progress or mark goals as mastered.
  • Enable interactive features that provide a sense of accomplishment and motivate users to continue.

Visual Aids and Resources:

  • Incorporate images and video modeling to demonstrate how to achieve goals.
  • Allow users to upload their own videos showcasing goal accomplishments.
  • Include a "Need Help?" link to connect users with external resources or YouTube tutorials for additional guidance.

Reminder System:

  • Implement a reminder text feature to help users remember their goals.
  • Utilize notifications or pop-ups to prompt users to take action and stay on track with their objectives.

Anticipated Platform / Tooling

The application will be available on Android and iOS by using the tools, React Native and Expo. We will be utilizing Visual Studio Code for our programming platform and also will be utilizing tools like Firebase, or a cloud service similar to, to have an online database to store the application and user information.

The application must be easy to understand and navigate and have plenty of pictures. Many autistic kids have a hard time learning and understanding if there are too many words.
